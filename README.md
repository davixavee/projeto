# projeto webcarros

<h2><n>O nosso trabalho se trata de um catálogo de automotores, como se fosse uma espécie de Webmotors.</n></h2>

<h3>Esse catálogo vai contar com varias categorias de automotores grandes e pequenos (motos, carros, furgões, caminhões, wagon, etc...).
Onde o dono do carro vai precisar preencher informações para catalogar seu carro, sendo elas, seu email, CPF, placa do veículo, documentação, número do chassis, ano de fabricação e modelo do automóvel. Cada CPF poderá ter, no máximo, 6 automóveis no site</h3>
